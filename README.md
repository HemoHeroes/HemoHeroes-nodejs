## How to contribute
1. Open a issue to we discussed about
2. Make fork of project
3. Create new Branch
4. Open a pull request

## Run in development
```bash
git clone https://github.com/HemoHeroes/hemoheroes-api.git
cd hemoheroes-api
npm run build-dev
npm run dev
```

## Run in production
```bash
git clone https://github.com/HemoHeroes/hemoheroes-api.git
cd hemoheroes-api
npm run build-prd
node ./bin/server.js
```
